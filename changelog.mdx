## Version History

Review the most recent updates below. Each release includes detailed changes grouped by category.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>
## New Features

- Added support for custom brand colors in themes, like `#3B82F6` for your projects
- Introduced `<Columns>` component for flexible grid layouts in docs

## Improvements

- Enhanced MDX parsing speed by 40% for larger documentation sites
- Better error messages when escaping special characters like `{` or `<`

## Bug Fixes

- Fixed nested code block rendering with 4 backticks
- Resolved heading hierarchy warnings in validation
</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>
## New Features

- New `<Tabs>` component for platform-specific instructions
- Integrated Lucide icons with auto-completion in editor

## Bug Fixes

- Corrected `<ParamField>` single-location attribute enforcement
- Patched Mermaid code block parsing (no more `<Mermaid>` component errors)

## Improvements

- Optimized image loading with required `width` and `height` attributes
</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>
## New Features

- Core MDX documentation engine with 15+ components
- YAML frontmatter validation for titles and descriptions
- Real-time preview for Ye Ye Documentation pages

## Breaking Changes

- Removed legacy H1 headings; always start with H2
- Enforced double quotes in all JSX attributes

## Initial Setup

- Full component library including `<Steps>`, `<CodeGroup>`, and `<Callout>`
</Update>
