## Overview

Ye Ye Documentation provides a centralized space to organize, manage, and publish your project's documentation. You create structured pages, embed interactive components, and customize the look to match your brand. This default space handles everything from quickstarts to API references with built-in MDX support for rich content.

<Callout kind="info">
  Ye Ye Documentation uses MDX, so you combine Markdown with JSX components for dynamic pages.
</Callout>

## Get Started in Minutes

Follow these steps to set up your first documentation site:

<Steps>
  <Step title="Create Your Space" icon="plus">
    Log in and select "New Documentation Space". Name it "Ye Ye Documentation".
  </Step>
  <Step title="Add Pages" icon="file-text">
    Create `introduction.mdx` as your landing page. Use the file explorer to organize sections.
  </Step>
  <Step title="Customize Theme" icon="settings">
    Edit `theme.config.js` to set your primary color to `#3B82F6`.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy with one click. Share the live URL with your team.
  </Step>
</Steps>

## Customization Options

Tailor Ye Ye Documentation to your needs across platforms.

<Tabs>
  <Tab title="Theme Config" icon="palette">
    Update your theme settings:

    <CodeGroup tabs="JavaScript,TypeScript">
    ```javascript
    // theme.config.js
    export default {
      color: '#3B82F6',
      logo: '/logo.svg',
      nav: [
        { title: 'Docs', href: '/docs' },
        { title: 'API', href: '/api' }
      ]
    }
    ```
    ```typescript
    // theme.config.ts
    const config = {
      color: '#3B82F6',
      logo: '/logo.svg',
      nav: [
        { title: 'Docs', href: '/docs' },
        { title: 'API', href: '/api' }
      ]
    } as const;
    export default config;
    ```
    </CodeGroup>
  </Tab>
  <Tab title="MDX Components" icon="code">
    Enhance pages with components like `<Callout>` and `<Steps>`:

    ```jsx
    <Callout kind="tip">
      Use `{brandColor}` for custom styles.
    </Callout>

    <Steps>
      <Step title="Step 1">Your content here.</Step>
    </Steps>
    ```
  </Tab>
</Tabs>
